# Fantasy Baseball Manager

A projection-driven fantasy baseball toolkit. The core of the system is a collection of statistical models that forecast player performance, giving you an edge in drafts, trades, and lineup decisions.

## Development

```bash
uv run pytest              # run tests
uv run ruff check src tests  # lint
uv run black src tests       # format
uv run ty check src tests    # type check
```
